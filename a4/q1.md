#### a) Define a page-replacement algorithm using this basic idea. Specifically address these problems:

**i) What the initial value of the counters is**

The initial value of all counters will be 0

**ii) When the counters are increased**

The counter of a frame will increase when a new page is swapped into it and becomes associated with the frame

**iii) When the counters are decreased**

The counter of a frame will decrease when one of the pages associated with it will not be used again and is swapped out

**iv) How the page to be replaced is selected**

The frame with the lowest counter will have its current page swapped out for the new page. If there are frames with the same lowest counter value, FIFO will be used to select which frame will have its contents swapped
